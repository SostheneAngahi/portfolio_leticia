Projet Réseau 1 – Cégep de Drummondville

Description : Configuration d’un réseau local avec un serveur, deux clients et un routeur Wi-Fi. IDE / Outils : Windows, configuration réseau, routeur domestique, tâches planifiées, antivirus

Projet Galaga NES – Cégep de Drummondville

Description : Recréation d’un jeu inspiré de Galaga sur la console NES. L’arrière-plan est fixe et utilise plusieurs motifs et attributs. IDE / Outils : Assembleur 6502, émulateur NES, gestion mémoire PPU/OAM, GitLab

Projet Final – Systèmes d’exploitation (Cégep de Drummondville)

Description : Installation et configuration d’un environnement multi-systèmes d’exploitation comprenant Windows 10/11 Professionnel et deux distributions Linux. Mise en place de GRUB personnalisé, gestion des utilisateurs (Jake, Jane, Jill et John), création de dossiers partagés sécurisés, configuration de permissions, intégration d’un système OPNsense en machine virtuelle, ainsi que sauvegardes complètes (GRUB, systèmes, VM et scripts). IDE / Outils : Windows, distributions Linux, GRUB, OPNsense (FreeBSD), machine virtuelle, scripts, outils de sauvegarde.
